# AzureImageAnalysis
This repo is a python script to connect to Azure Cognitive services, loop through a list of URLs and extract image data to files for further analysis.

## Setup

This script requires that you have access to a Cognitive Services resource in Azure.  A walkthrough on how to create an Azure Cognitive Services resource is availalbe [here](https://learn.microsoft.com/en-us/azure/cognitive-services/cognitive-services-apis-create-account?tabs=multiservice%2Canomaly-detector%2Clanguage-service%2Ccomputer-vision%2Cwindows)
